# EnlazadosTWBE
EnlazadosTW es un ecosistema de software orientado a la interoperabilidad humana y de datos en el ámbito de la salud integral y la educación especial. El sistema se compone de una API construida con Spring  y PostgreSQL con seguridad basada en roles mediante JWT, y una aplicación móvil multiplataforma desarrollada con Angular e Ionic Framework.
